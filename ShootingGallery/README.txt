Aim Simulator

The program uses SFML to provide the graphics of the game.
To run the program you will need to compile the c++ file in the terminal.
Afterwards, it is just a matter to run the compiled program.

Instructions:

Game starts of at the menu where the choice are choosing the level of
difficulty and the exit which closes the button.
Once the level of difficulty is choose, the player only has to click with their mouse
the targets that appear randomly in the screen.
The player is given 60 min to hit the targets=, and after completed they will be shown the results:
Precision
Score
Raction Time

The player can then decide if they want to restart back to the menu to play again or just exit the game
completely.
